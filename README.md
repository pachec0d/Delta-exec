## Installation
### For Regular Users
1. Download and extract the archive
2. Run `DeltaRoblox.exe` as Administrator
3. Log in to your Roblox account
4. Adjust graphics and performance settings
5. Enjoy smoother gameplay

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| Failed to launch           | Run as Administrator                          |
| Low FPS                    | Lower graphics quality or enable Performance Mode |
| Black screen               | Switch to Borderless Windowed mode            |
| Antivirus warning          | Add the folder to exclusions                  |
| Login / connection issues  | Clear cache or check internet connection      |
